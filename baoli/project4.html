酒店项目合作模式

  • 委托管理
  • 租赁经营
  • 开业前技术顾问咨询

保利酒店的委托管理，核心原则采用国际通行的惯例和标准，具体条款则做了适应性调整，赋予业主更多权力，确保和业主建立互信、双赢的长期合作关系。
委托管理的核心原则是：

管理公司相对独立自主地负责经营管理，业主负责管理资产审批和预算，对管理公司进行监督。即业主保留对酒店资产的完全控制权，并通过审核和批准酒店预算，来管理酒店的资金使用状况，
并监督管理公司按照合同约定进行经营管理。而管理公司确保基本上按照自己的方式，发挥专业能力及资源共享，为业主获取最大回报。

对于有意向和保利酒店合作的业主，我们会在确认酒店市场前景、合作方式、酒店功能布局及建造标准后，向业主递交《合作意向协议》（基本合作意向条款），与业主就主要合作意向达成共识
在此基础上，我们将与业主讨论并签署《开业前技术顾问咨询协议》、《保利酒店委托管理协议》，确定所有相关合作细节。

如果业主希望获取固定收益，保利酒店也采用租赁经营的合作方式。

我们可以根据业主的需求、项目的状况采用灵活的租赁经营方式。我们可以租赁一家运营中的酒店，也可以投入资金进行装修后租赁经营。 如果是投入装修，酒店投资回报需要有更加清晰的市场预期，我们希望酒店处于核心城市，消费能力强。

保利酒店已运营及在建的项目超过50家，绝大部分是自己投资运营的五星级酒店。

这使保利酒店具有丰富的筹建管理经验，在成本控制、进度管理、新设备新技术的运用等方面具有独到的经验。集团设有专职技术发展总监、项目管理总监和设计顾问， 他们会同管理公司各职能总监，作为业主的顾问专家，在项目的规划、设计、兴建期间提供服务和协助，以保证酒店顺利开业。

我们提供的开业前顾问咨询工作包括：

[ 周密的市场分析以确定酒店市场定位 ]
[ 在详尽的市场分析基础上确定酒店设计任务书 ]
[ 推荐合适的设计师参与酒店设计 ]
 [ 资深管理团队反复探讨酒店功能合理性 ]
[ 资深管理团队为酒店制定所有设备标准和技术标准 ]
[ 通过保利集团采购中心，为酒店提供物品和设备采购咨询或者服务 ]
 [ 为业主制定精确的开业预算 ]
[ 安排周密的开业进程表和筹建程序 ]
[ 协调集团所有资源给予开业各项支持 ]

合作服务内容

项目管理

在多年酒店筹建过程中，保利酒店形成了一整套专业化的流程管理制度，如《项目管理手册》、《酒店工程建设及投资控制流程》、《施工图纸内部会审管理制度》、《酒店项目质量监督小组工作制度》、《现场工程管理办法》、《招标管理制度》、《甲供材料管理制度》等，并且在实际操作过程中取得了较好的效果，在项目的管理过程中，将根据现场实际情况对各种流程进一步优化，保证工程达到业主要求的工程管理控制目标。

  • 预算

  • 进度

  • 核验

  • 分析

  • 调试

  • 验收

建议工程总投资，使工程各阶段、各单项工程费用支出与形象进度相协调；使项目最终投资目标的变化控制在既定预算目标以内。

根据业主进度目标，确定主要施工节点控制目标，制定交付使用的项目完工倒计时施工总计划表，严格实施计划进度管理，实现本工程按期顺利竣工交付使用的战略目标。

负责审核技术标准、方案设计、施工图设计，对于设计中存在的差错及时责令设计单位改正，严把质量关。

根据工程进展向业主上报关于工程进度、投资、质量等方面的综合评估分析报告，并对存在问题作出分析和方案建议。

负责协调监理进行项目调试大纲的编制工作，并组织实施项目各系统的单机调试、联动调试与总调试工作。

工程具备竣工验收条件时，协助业主组织监理、监管单位、质量监督、承包商等方面部门代表组成交工验收组，进行工程竣工验收。交付运营团队负责酒店正常开业运营。
